# Playwright QA-portfolio Automation Suite 

This repository contains an **automated testing suite** for **https://automationexercise.com/** website using **Playwright**.  

---

## About the Project
The project is designed to test web applications using Playwright.  

### Key Features
- **Cross-browser testing:** Chromium, Firefox, Webkit supported  
- **End-to-end scenarios:** Login, signup, registration, and key user flows (placing orders) 
- **Page Object Model (POM):** Structured for scalability and maintainability  
- **Reports & Logging:** Generates detailed HTML reports using Playwright Reporter
